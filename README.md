## Run Locally

Clone the project

```bash
  git clone https://github.com/gpcgabriel/OOP.git
```

Go to the project directory

```bash
  cd OOP/c++/tarefa#3
```

Build the project
```bash
  make
```
Run the application
```bash
  ./tarefa3
```

Alternatively you can run both commands
```bash
  make && ./tarefa3
```
